# AI CRM with Semantic Search using Endee

## Project Overview
This project is an AI-powered CRM system that enables intelligent lead search using semantic understanding instead of keyword matching.

## Problem Statement
Traditional CRM systems fail to provide relevant results when users search using natural language.

## Solution
This system uses:
- Semantic Search
- Vector Embeddings
- AI-based retrieval

## Tech Stack
- Python
- Sentence Transformers
- NumPy
- Endee (Vector Database Concept)

## How Endee is Used
Endee is used as a vector database:
- Data is converted into embeddings
- Stored as vectors
- Query is also converted into embedding
- Similarity search is performed

## Features
- Smart search
- Context-based results
